Mission

Effectuer un audit physique complet du moteur de simulation scientifique.
L’objectif est de vérifier que toutes les grandeurs calculées respectent les lois physiques fondamentales, les unités cohérentes et les invariants dynamiques.

Cet audit doit identifier :
	•	normalisations implicites
	•	incohérences dimensionnelles
	•	erreurs dans l’Hamiltonien
	•	défauts dans les intégrateurs numériques
	•	violations des lois de conservation

1. Reconstruction automatique des unités physiques

Scanner tous les modules et reconstruire les dimensions physiques de chaque variable.

Détecter automatiquement les grandeurs suivantes

energy
time
length
momentum
mass
density
pressure
charge
action
temperature

Associer à chaque variable une dimension :

Quantité	Dimension
énergie	ML²T⁻²
vitesse	LT⁻¹
densité	ML⁻³
pression	ML⁻¹T⁻²

Génération d’un graphe dimensionnel

Construire un graphe reliant :

variables

unités

Sortie :

logs/dimensional_graph.json

Identifier automatiquement :

dimension_mismatch
unit_loss
implicit_scaling

2. Reconstruction automatique de l’Hamiltonien

Identifier dans le code les expressions correspondant à :

H
L
S

où :

H = Hamiltonien
L = Lagrangien
S = action

Reconstituer la forme effective :


H = T + V


avec :

T = énergie cinétique
V = potentiel

Vérifications obligatoires
	1.	positivité de l’énergie cinétique


T \ge 0

	2.	symétrie hermitienne


H = H^\dagger

	3.	invariance sous transformations physiques

selon le modèle :

translation
rotation
gauge symmetry

Produire :

logs/hamiltonian_reconstruction.json

Chaque anomalie doit contenir :

file

4. Audit des intégrateurs numériques

Identifier les intégrateurs utilisés :

Euler
Runge-Kutta
Verlet
Leapfrog
Symplectic integrators
Monte Carlo updates

Vérifier :

ordre de convergence
stabilité numérique
erreur globale

Test de symplecticité

Pour les systèmes hamiltoniens :


\frac{dH}{dt} = 0


Un intégrateur correct doit approximer cette propriété.

Mesurer :

ΔH(t)

sur toute la simulation.

Produire :

logs/integrator_energy_drift.csv
