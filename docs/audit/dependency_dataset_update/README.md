# Traçabilité mise à jour dataset `nx47-dependencies`

Ce dossier est destiné à être généré/mis à jour par l'agent Replit avant push vers le dataset Kaggle.

## Fichiers
- `dataset_current_contents.md`: snapshot du contenu déjà présent dans le dataset (fourni par l'utilisateur).
- `requirements_v7_python_pure.txt`: liste des dépendances Python V7 à embarquer (mode Kaggle 100% Python).
- `missing_from_dataset.md`: écart entre le contenu actuel et la liste cible.
- `update_commands.md`: commandes reproductibles pour générer le bundle local et préparer le push dataset.

## Objectif
- Conserver l'existant.
- Ajouter les dépendances manquantes pour V7.
- Journaliser ce qui a été ajouté/modifié pour audit.
