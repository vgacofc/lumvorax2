# NQubit_v6 — Single-cell Kaggle/Catlin Kernel

Cette version est conçue pour Kaggle/Catlin en **une seule cellule**.

## Fichier unique
- `nqubit_v6_catlin_kaggle_kernel.py`

Ce fichier contient **tout**:
- détection environnement/GPU/CPU,
- collecte métriques système,
- capture entropy/jitter hardware,
- stress,
- manifest build/verify,
- rapport final.

## Utilisation
Copier-coller tout le contenu de `nqubit_v6_catlin_kaggle_kernel.py` dans la cellule 1 d’un notebook Kaggle, puis exécuter.

## Sorties
Les artefacts sont produits dans `results/` du répertoire courant du notebook.
